# Python Para Pentest
## Baseado nos seguintes livros: 


### Use a Cabeça: Python - 3ª Edição - Paul Barry
https://amzn.to/4rmlF5U


No Repositório **'head-first/Python'** = Use a Cabeça Python (do autor Paul Barry)
Respostas de alguns exercícios e exemplos de códigos, adaptados por mim. 

Esse livro é bom para aqueles que querem aprender do básico ao avançado em Python, porém é importante saber alguma outra linguagem de programação ou pelo menos ter uma base de Lógica e Algoritmo de Programação, sem isso pode ter um pouco mais de dificuldades.

-----------------------------------------------------------------

 ### Python para Pentest - Daniel Moreno
https://amzn.to/45mmbIB

No Repositório **'DHM'** = Python Para Pentest (do autor Daniel Henrique Moreno)
Códigos adaptados do Python2 para o Python3, por mim.

Esse livro dá uma introdução da Linguagem Python, porém está desatualizado na versão 2 da linguagem, o que pode dificultar a execução dos exercícios para quem está iniciando na linguagem e na área de Desenvolvimento de Softwares para Pentest e Segurança da Informação.
Porém as ferramentas introduzidas ao leitor aqui são importantes e os conceitos iniciais da linguagem ajudam, se não houver outra alternativa.


-----------------------------------------------------------------

  ### Black Hat Python - 2ª Edição - Justin Seitz e Tim Arnold
https://amzn.to/45bdShs

No Repositório **'BHP'** = Black Hat Python 

Esses exercícios e códigos propostos no livro trazem as ferramentas prontas para uso.
